# STEM Hub Scotland
Website for students, teachers, event partners and volunteers. <br>
That either want to get into or to further their knowledge of STEM base extra curricular activities e.g. vex robotics, stem racing.

### Overview
- The goal is to build a platform where people interested in stem can find information and support for roles like students, teachers, volunteers and event organisers across many different stem competitions and events. <br>
- A key note is, this platform is not ment to replace the systems already in place by each individiual orginisation like Vex robotics but instead have a central place where information can be shared about these events and opertunities, with a particular focus on getting all the information new teachers need who want to start stem activities at thier schools or clubs.

### Roles (features)
Teachers find:
- Competitions
- Workshops
- Resources
- Training events
- Local contacts and supports (other teachers in the area)

Students find:
- Opportunities
- Communities
- Other teams
- Inspiration
- Forums to share information and tips

Volunteers find:
- Ways to help
- Upcoming events
- Contacts

Event Orginisers find:
- volunteers
- Help teachers
